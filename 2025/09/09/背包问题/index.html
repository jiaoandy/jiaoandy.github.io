背包问题

背包问题是动态规划中非常经典的问题，主要分为以下几种类型：

  1. 0-1背包问题：每种物品只能选择一次，目标是最大化背包中的总价值而不超过容量限制。
  2. 完全背包问题：每种物品可以选择多次，目标是最大化背包中的总价值而不超过容量限制。
  3. 多重背包问题：每种物品有固定的数量，目标是最大化背包中的总价值而不超过容量限制。
  4. 分组背包问题：物品被分为若干组，每组物品只能选择一个，目标是最大化背包中的总价值而不超过容量限制。

动态规划解题思路

动态规划的核心思想是将问题分解为子问题，逐步解决这些子问题，并利用已解决的子问题的结果来构建最终解决方案。核心是找到状态转移方程，我们可以分几步思考：

  1. 题目要求什么？
  2. 什么性质？
  3. 在得到的状态转移方程中，哪些是已知的，哪些是未知的？
  4. 未知的是否可以通过已知的状态转移得到？
  5. 如何优化？（可最后完成代码后优化）
    1. 空间优化：可以使用滚动数组来优化空间复杂度，将二维数组优化为一维数组。
    2. 时间优化：可以使用二进制优化来优化时间复杂度，将多重背包问题优化为0-1背包问题。
    3. 状态压缩：可以使用位运算来压缩状态，将二维数组优化为一维数组。

0-1背包问题

问题描述

有N个物品，每个物品的重量为w[i]，价值为v[i]，背包容量为V，求怎么放能让背包装入物品的价值最大化。
输入格式:
第一行两个整数，N，V，用空格隔开，分别表示物品数量和背包容积。
接下来有 N行，每行两个整数 v[i],w[i]，用空格隔开，分别表示第 i件物品的体积和价值。

输出格式:
输出一个整数，表示最大价值。

数据范围
0 < N,V≤1000

0 < v[i],w[1] ≤ 1000

输入样例
4 5
1 2
2 4
3 4
4 5
输出样例：
8

分析

如果使用暴力算法，即遍历每个物品，很容易超时。因此我们需要用动态规划的思想解决这个问题。
开始思考：

  1. 题目要求的是什么？ 最大价值
  2. 什么性质？ 每个物品只能选择一次
  3. 在得到的状态转移方程中，哪些是已知的，哪些是未知的？ 已知物品的重量和价值，未知最大价值
  4. 未知的是否可以通过已知的状态转移得到？ 可以，通过选择第i件物品和不选择第i件物品的状态转移方程得到。
