    方法二：单调栈+贪心+二分

    思路

    我们维护一个数组d，d[i]表示长度为i的最长上升子序列的最后一个元素的最小值。接着，遍历数组 a[]，如果a[i]>=d[top]，那么a[i]压入d[]中，top++。
    否则，用a[i]替换d[j]，其中0<j<top且d[j-1]<a[i]<d[j]。

    举个例子：

    数列：[3, 1, 2, 4, 1, 5]

    步骤：

    s = [3]

    遇到1：替换3 → s = [1]
    虽然序列长度没变，但让后面的数字更容易加入

    遇到2：大于1 → s = [1, 2]

    遇到4：大于2 → s = [1, 2, 4]

    遇到1：替换第一个1 → s = [1, 2, 4]
    序列没变，但为后面做准备

    遇到5：大于4 → s = [1, 2, 4, 5]

    最终长度：4

    重点：在替换的过程中，d[]数组的长度不会变化，仅仅是数组的元素发生变化。而如果我们把小的值压入（替换）d[]中，那么后面的数字就更有机会加入序列中，从而增加序列的长度。
